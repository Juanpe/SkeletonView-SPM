The goal of this repository is to define [SkeletonView](https://github.com/Juanpe/SkeletonView) as a **XCFramework**. Now, you can use this repo to install it. To get more details about the library and its features, please go to [the original repo](https://github.com/Juanpe/SkeletonView).

**Supported OS & SDK Versions**

* iOS 9.0+
* tvOS 9.0+
* Swift 5.3
